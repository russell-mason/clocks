Clocks is a simple memory game against the clock!

All you have to do is remember the time.

You can set various options that make the game more difficult, such as how many clock faces you see at once, whether the time includes seconds, and how long you have to remember, and enter, your guess.

Scores are based on how difficult the game is, and how long you take to respond.

Help

Although the game is simple to pick up, the following sections fully describe what options are available, and how to play the game.

Theme

Determines whether the application uses a light or dark background.

Number of clocks

Determines whether one, two, or four, clocks are displayed at once.

Because it's harder to remember multiple clocks in a single game than it is to remember a single clock in multiple games, scores are weighted higher for additional clocks.

Clock face

Show am/pm indicators

Determines whether AM and PM indicators are shown.

When this option is not checked, no AM/PM indicators will be shown. Twelve/twenty-four hour times are seen as equivalent. e.g. If the clock shows "3:45" and you enter 03:45 or 15:45, both will be treated as correct.

When checked, AM/PM indicators will be shown. Twelve/twenty-four hour times are seen as distinct and you must enter the matching value. e.g. If the clock shows "3:45" with a "PM" indicator, you must enter 15:45 to be correct, entering 03:45 will be treated as incorrect.

Higher scores are given when indicators are shown.

Show second hand

Determines whether a second hand is shown on the clock.

A clock with no second hand is equivalent to remembering a four digit number.

A clock with the second hand is equivalent to remembering a six digit number.

Higher scores are given for times when the second hand is shown.

Show markers

Determines which markers are shown around the clock face.

The fewer markers shown, the higher the score given.

Countdown timers

Memorize

Determines the amount of time you're given to memorize the time.

Recall

Determines the amount of time you're given to recall the time.

Both allow a selection from 5 seconds to 60 seconds, in 5 second increments.

Higher scores are given if there is still time left on the countdown when you continue to the following screens.

Once you've selected the options you want, it's time to play.

Click the "next" button; don't worry, the game won't start immediately.

Get Ready

This allows you to dictate when the game starts.

When you're ready, click the "play" button and the game will proceed to the "Memorize" screen, and the countdown will begin immediately.

Memorize

This screen will show one or more clocks using the chosen options.

Each clock has a corresponding number so that when it comes time to recall your guess, you know which time to assign to which clock.

Once memorized, you can click the "stop" button to continue, or let the time run out.

When the counter reaches zero, the game will automatically proceed to the "Recall" screen, and the next countdown will begin immediately.

Tip:

  • Proceeding to the next screen before the time runs out will result in a higher score.

Recall

In this screen you need to recall the times you were just shown.

Click on the clock with the number you wish to enter a time for (the first clock will be selected by default), and use the number pad to enter the time. You don't need to include ":" separators, they will automatically be added and shown against the currently selected clock. You must enter four digits or six digits, depending on the selected options, i.e. 6:18 must be entered as 0618, 9:23:51 must be entered as 092351.

You can freely switch between clocks at any time. If the current value in invalid it will be shown greyed in the number pad display and no time will be shown on the clock face.

Once you've entered all times, you can click the "check" button to see how well you've done, or let the time run out.

When the counter reaches zero, the game will automatically proceed to the "Scores" screen.

Tips:

  • Proceeding to the next screen before the time runs out will result in a higher score.
  • When "Show second hand" is selected, you can still score points with a partially correct answer, i.e. you get the hours and minutes correct, but the seconds wrong.

In addition to clicking on the clock and number pad, you can also use the keyboard, see "Using the keyboard" below.

Scores

This screen shows the results of the completed game, along with the total scores for all the games you've played.

You can continue with one of the following:

  • Intro: Return to this screen, so you can re-read the instructions.
  • Options: Return to the "Options" screen, so you can change options prior to playing another game.
  • Play again: Return to the "Get Ready" screen, so you can play another game with the same options.

Using the keyboard

  • Pressing the "Enter" key is the same as clicking the button on the right-hand side of the footer.
  • When scores are being shown, "Enter" will activate the right-most button, i.e. "play again"
  • On the "Recall" screen, focus is automatically set to the number pad, so you can just start typing to enter a number.
  • Pressing the "Backspace" key will delete the last number entered.
  • Pressing the "Esc" key or the "c" key will clear the entire entry.
  • When multiple clocks are displayed, you can switch between them using "Ctrl" ("control" on a Mac) and the corresponding clock number,
    e.g. Ctrl+1, Ctrl+2, Ctrl+3, or Ctrl+4.
  • Focus will automatically return to the number pad and you can continue to enter/edit a number as normal.
